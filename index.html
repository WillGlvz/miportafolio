Acerca de mi

Mi nombre es Willian Eduardo Vásquez Gálvez, a la fecha de hoy, Febrero del 2021, poseo 25 años de edad, estoy cursando mi quinto año en la carrera de ingeniería en ciencias de la computación, Universidad Don Bosco. Mis estudios de bachiller los llevé a cabo en el Instituto Técnico Ricaldone, durante tres años estudie la especialidad de Técnico en ingeniera de sistemas informáticos. He desarrollado una variedad de software para fines academicos obteniendo buenos resultados finales, en el año 2015 desarrollé un sitio web para una empresa de localidad Salvadoreña, en el año 2017 desarrollé un sistema de escritorio para otra empresa local y tengo experiencia en manteniento informático. Me gusta aprender de los demás y que los demás aprendan de mi.

2015

Sistema web de control para depósito fiscal

Un sistema web desarrollado para una empresa local, el sistema lleva un control de todo tipo de productos que entran por medio de aduana, lleva todo el control de la logistica que se requiere para que un producto entre en condiciones al país. El sistema fue desarrollado en HTML5, CSS3 y Bootstrap Framework para la parte del Frontend. La parte del Backend esta compuesta por PHP puro, javascript, Ajax y Json. Se usó MySql como base de datos del sistema. El sistema web no fue públicado por problemas de hosting.
